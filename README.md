# Driving Omnichannel Conversion Through Clickstream Intelligence

## Link to Project Portfolio Story: *Medium article in Progress*

## Executive Summary

Ads. How good are they, actually? This project will attempt to analyze the behavior of pregnant women while shopping for certain products and apply statistics, holistic-yet-meticulous data science methodologies, and cutting-edge decision science thinking to drive the optimization of advertisements.

## Credit for the Dataset:

UC Irvine Machine Learning Repository

## Tools & Skills Used

* Python (programming language): Pandas, NumPy, Matplotlib, Seaborn, SciPy, Scikit-learn
* Applied Statistics:
  Hypothesis Testing - ANOVA, Chi-Squared Test of Independence
* Machine Learning:
  Preprocessing Pipelines & Transformers, Logistic Regression, K-Means Clustering (Unsupervised Learning)
